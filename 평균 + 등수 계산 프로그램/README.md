# 평균 + 등수 계산 프로그램 (동적 2차원 배열)

## 1. 개요
학생 수와 과목 수를 입력받아 각 학생의 과목별 점수를 저장한 뒤,  
학생별 평균을 계산하고 평균 점수를 기준으로 등수를 출력하는 C 프로그램입니다.

- 입력 기준: **학생 단위**
- 저장 구조: `arr[학생][과목]`
- 평균 계산: 각 학생의 과목 점수 합 / 과목 수
- 등수 계산: 평균이 더 높은 학생 수 + 1

---

## 2. 실행 방법

### 입력 예시
```text
과목 수 입력 : 2
학생 수 입력 : 3
학생 0 점수 ------------
과목 0 점수 입력 : 100
과목 1 점수 입력 : 90
학생 1 점수 ------------
과목 0 점수 입력 : 90
과목 1 점수 입력 : 90
학생 2 점수 ------------
과목 0 점수 입력 : 100
과목 1 점수 입력 : 80
```

### 출력 예시
```text
========== 학생 평균 ==========
학생 0 평균 : 95.00, 1등
학생 1 평균 : 90.00, 2등
학생 2 평균 : 90.00, 2등
```
---

## 3. 메모리 구조
```text
arr
 ├─ arr[0] → int[subjects]
 ├─ arr[1] → int[subjects]
 └─ arr[2] → int[subjects]
```

---

## 4. 주요 함수 역할

- `student_average`  
  각 학생의 과목 점수를 합산하여 평균을 계산하고 평균 배열에 저장합니다.

- `student_rank`  
  학생 평균 배열을 기반으로 상대 비교를 수행하여 등수를 계산합니다.


---

## 5. 등수 계산 로직

등수는 정렬을 사용하지 않고, 각 학생의 평균보다 높은 평균을 가진 학생의 수를 세는 방식으로 계산합니다.

```text
rank = 1 + (나보다 평균이 높은 학생 수)
```

--- 

## 6. 메모리 관리

본 프로그램은 동적 할당된 2차원 배열을 사용하므로, 할당 순서의 역순으로 메모리를 해제하여 메모리 누수를 방지합니다.

- 학생별 점수 배열 해제
- 포인터 배열 해제

- C 언어
- 동적 메모리 할당 (`malloc`, `free`)

- 배열 기반 평균 및 등수 계산 로직

---
## 8. 시간 복잡도

- 평균 계산: `O(학생 수 × 과목 수)`
- 등수 계산: `O(학생 수²)`
