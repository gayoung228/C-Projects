# 📚 도서 관리 프로그램 (멀티 파일 버전)

C 언어로 작성한 **콘솔 기반 도서 관리 프로그램**입니다.  
이번 버전은 기능별로 **소스 파일(.c)**과 **헤더 파일(.h)**을 나누어 관리하도록 구성했습니다.

---

## ✨ 주요 기능

- 책 정보 등록
  - 책 이름, 저자, 출판사 입력
- 책 검색
  - 제목 / 지은이 / 출판사로 검색
  - 부분 문자열 포함 검색 (예: `one` → `Everyone`에서도 검색됨)
- 책 대출
  - 책 번호로 대출 처리
  - 이미 대출된 책은 대출 불가
- 책 반납
  - 책 번호로 반납 처리
  - 이미 반납된 책은 안내 메시지 출력

---

## 📁 프로젝트 구조

예시 폴더 구조:

```text
도서 관리 프로그램
└── 파일 나눈 버전
    ├── 소스 파일
    │   ├── BookManageProgram_file.c   // main 함수 (프로그램 시작점)
    │   ├── BookManageProgramFunc.c    // 도서 관리 관련 함수들 (추가/검색/대출/반납)
    │   └── BookManageProgramStr.c     // 문자열 비교 함수(compare)
    │
    └── 헤더 파일
        ├── BookManageProgramFunc.h    // 도서 관리 함수 선언
        └── BookManageProgramStr.h     // 문자열 관련 함수 선언
