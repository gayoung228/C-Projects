# 도서 관리 프로그램 (C 언어 콘솔 프로젝트)

간단한 콘솔 기반 **도서 관리 프로그램**입니다.  
책을 추가하고, 제목/지은이/출판사로 검색하고, 대출/반납까지 관리할 수 있습니다.

---

## 📚 기능 소개

### 1. 책 새로 추가하기
- 메뉴에서 `1`을 선택하면 새 책 정보를 입력할 수 있습니다.
  - 책 이름
  - 저자 이름
  - 출판사 이름
- 추가된 책은 내부 배열에 순서대로 저장되며, 초기 상태는 **대출 가능(borrowed = 0)** 입니다.

---

### 2. 책 검색하기
- 메뉴에서 `2`를 선택하면 검색 메뉴로 이동합니다.
- 검색 옵션:
  1. 제목으로 검색
  2. 지은이로 검색
  3. 출판사로 검색
- 검색어는 **부분 일치**로 동작합니다.
  - 예: `str1 = "Everyone_C_Programming"`, `str2 = "one"` 이면 일치로 판단  
- 검색 결과에는 다음 정보가 출력됩니다.
  - 번호 (배열 인덱스)
  - 책 이름
  - 지은이
  - 출판사
  - 대출 여부 (`대출 가능` / `대출 불가능`)

---

### 3. 책 빌리기
- 메뉴에서 `3`을 선택하면 **대출할 책 번호**를 입력합니다.
- `borrowed[번호]` 값으로 대출 상태를 관리합니다.
  - `0` → 대출 가능
  - `1` → 이미 대출됨
- 상태에 따라 다음과 같이 처리합니다.
  - 대출 가능 → `성공적으로 대출되었습니다!`
  - 이미 대출된 책 → `이미 대출된 책입니다.`

---

### 4. 책 반납하기
- 메뉴에서 `4`를 선택하면 **반납할 책 번호**를 입력합니다.
- 상태에 따라 다음과 같이 처리합니다.
  - 이미 반납된 상태(`borrowed == 0`) → `이미 반납된 책입니다.`
  - 대출 중인 상태(`borrowed == 1`) → `성공적으로 반납되었습니다!` 이후 `borrowed = 0`으로 변경

---

### 5. 프로그램 종료
- 메뉴에서 `5`를 선택하면 `while (1)` 루프를 빠져나오며 프로그램이 종료됩니다.

---

## 🧩 사용한 주요 함수들

### `int add_book(...)`
- 새로운 책 정보를 입력받아 각각의 배열(`book_name`, `auth_name`, `publ_name`)에 저장합니다.
- `borrowed[*num_total_book] = 0` 으로 초기화하여 **대출 가능 상태**로 설정합니다.
- `num_total_book` 값을 1 증가시킵니다.

### `int search_book(...)`
- 검색 옵션을 입력받은 뒤, 검색어를 기준으로 책을 찾습니다.
- 일치하는 책이 있으면 `print_book_status()`를 호출해 상세 정보를 출력합니다.
- 검색 결과가 하나도 없으면 `검색 결과가 없습니다.` 출력.

### `int search_option()`
- 검색 기준(제목 / 지은이 / 출판사)을 선택하는 메뉴를 출력하고 사용자의 선택 값을 반환합니다.

### `int compare(char *str1, char *str2)`
- **부분 문자열 포함 여부**를 확인하는 함수입니다.
- `str2`가 `str1` 안에 포함되어 있으면 `1`, 아니면 `0`을 반환합니다.

### `void print_book_status(...)`
- 특정 인덱스의 책 정보를 다음 형식으로 출력합니다:
  - `-> 번호 : N // 책 이름 : ... // 지은이 : ... // 출판사 : ... // 대출여부 : ...`

### `char* borrowed_status(int id, int* borrowed)`
- 해당 책의 대출 상태를 문자열로 반환합니다.
  - `대출 가능`
  - `대출 불가능`

### `int borrow_book(int* borrowed)`
- 입력받은 책 번호의 `borrowed` 상태를 보고 대출 처리 또는 에러 메시지를 출력합니다.

### `int return_book(int* borrowed)`
- 입력받은 책 번호의 `borrowed` 상태를 보고 반납 처리 또는 에러 메시지를 출력합니다.
