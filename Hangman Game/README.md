# 🎮 Hangman Game (C Language)

> C 언어의 핵심 개념을 학습하고 적용하기 위해 구현한  
> **콘솔 기반 행맨(Hangman) 게임 프로젝트**입니다.

---

## 📌 프로젝트 개요

이 프로젝트는 **C 언어로 구현한 콘솔 기반 행맨 게임**입니다.  
프로그램은 단어 목록 중 하나를 **무작위로 선택**하고,  
사용자는 주어진 힌트를 바탕으로 알파벳을 하나씩 입력하여 단어를 맞히게 됩니다.

본 프로젝트는 **배열, 문자열, 구조체, 조건문, 반복문, 함수 분리** 등  
C 언어의 기본이 되는 요소들을 실제 게임 로직에 적용하는 것을 목표로 제작되었습니다.

---

## 🕹️ 게임 진행 방식

1. 단어 목록에서 하나의 단어가 랜덤으로 선택됩니다.
2. 선택된 단어는 `_`(언더바) 형태로 가려진 상태로 출력됩니다.
3. 사용자는 한 번에 **알파벳 한 글자**를 입력합니다.
4. 입력한 글자가 단어에 포함되어 있으면 해당 위치에 표시됩니다.
5. 단어에 없는 글자를 입력할 경우 시도 횟수가 1 감소합니다.
6. 모든 글자를 맞히면 **게임 승리**,  
   제한된 시도 횟수를 초과하면 **게임 오버**가 됩니다.

---

## ✨ 주요 기능

- `rand()` / `srand(time(NULL))`를 이용한 단어 랜덤 선택
- 구조체(`struct`)를 활용한 단어–힌트 데이터 관리
- 알파벳 입력 유효성 검사
- 이미 입력한 알파벳 중복 입력 방지
- 현재 단어 상태 실시간 출력 (`_` / 맞춘 문자)
- 실패 횟수에 따른 행맨 ASCII 아트 출력
- 승리 / 패배 조건 자동 판별

## 🛠️ 사용 기술

### Language
- **C**

### Libraries
- **`stdio.h`** : 콘솔 입출력 함수 사용 (`printf`, `scanf`)
- **`stdlib.h`** : 난수 생성 및 기본 유틸리티 함수 사용 (`rand`, `srand`)
- **`string.h`** : 문자열 처리 함수 사용 (`strlen`, `strcmp`)
- **`time.h`** : 난수 시드 설정을 위한 시간 함수 사용 (`time`)
- **`ctype.h`** : 문자 처리 함수 사용 (소문자 변환 및 입력 검사)
- **`stdbool.h`** : `bool` 타입 사용을 위한 라이브러리

---

## ⭐ 핵심 기술 (Key Concepts)

- **랜덤 로직 설계**
  - `rand()`와 `srand(time(NULL))`를 이용해 실행 시마다 다른 단어 선택

- **구조체 기반 데이터 관리**
  - `struct`를 사용해 단어와 힌트를 하나의 데이터로 관리

- **문자열 상태 관리**
  - `_`로 초기화된 배열을 통해 맞춘 글자만 점진적으로 공개

- **입력값 검증 및 예외 처리**
  - 알파벳 이외 입력 차단
  - 이미 입력한 문자 중복 처리 방지

- **게임 상태 제어**
  - 시도 횟수(`tries`)에 따른 게임 흐름 제어
  - 승리/패배 조건 명확히 분리

- **함수 분리를 통한 역할 분담**
  - 출력 로직(`displayWord`, `displayHangman`)과 게임 로직 분리
  - 코드 가독성과 유지보수성 향상
