# 🎲 Snake and Ladder Game (C Language)

> 주사위를 굴려 이동하며 사다리는 올라가고, 뱀은 내려가는  
> **고전 보드게임 ‘스네이크 앤 레더(Snake and Ladder)’를 C 언어로 구현한 콘솔 게임 프로젝트**입니다.

---

## 📌 프로젝트 개요

본 프로젝트는 **C 언어로 구현한 콘솔 기반 스네이크 앤 레더 게임**입니다.  
두 명의 사용자가 번갈아 가며 주사위를 굴려 말을 이동시키고,  
사다리와 뱀의 규칙에 따라 위치가 변경됩니다.

게임의 목표는 **가장 먼저 100번 칸에 정확히 도착하는 것**이며,  
주사위, 턴 제어, 배열, 조건문, 랜덤 함수 등을 활용하여  
보드게임의 기본 로직을 구현하는 데 초점을 두었습니다.

---

## 🕹️ 게임 규칙

1. 게임은 **2명의 사용자**가 참여합니다.
2. 각 턴마다 사용자는 **엔터 키를 눌러 주사위를 굴립니다**.
3. 주사위에 나온 숫자만큼 말이 앞으로 이동합니다.
4. 이동한 칸에:
   - **사다리**가 있으면 위로 올라갑니다.
   - **뱀**이 있으면 아래로 내려갑니다.
5. 100번 칸을 **초과하면 이동할 수 없으며**, 해당 턴은 무효가 됩니다.
6. **가장 먼저 100번 칸에 도착한 플레이어가 승리**합니다.

---

## 🧩 게임 구성 요소

- **보드(Board)**  
  - 1부터 100까지의 번호가 있는 정사각형 보드
  - 출력 시 지그재그 형태로 표시

- **주사위(Dice)**  
  - 1부터 6까지의 난수를 생성하여 이동 거리 결정

- **플레이어(Player)**  
  - 두 명의 플레이어(P1, P2)
  - 현재 위치를 전역 변수로 관리

- **뱀과 사다리(Snakes & Ladders)**  
  - 특정 칸에 도착 시 추가 이동 발생
  - 양수 값 → 사다리 (위로 이동)
  - 음수 값 → 뱀 (아래로 이동)

---

## ✨ 주요 기능

- `rand()` / `srand(time(NULL))`를 이용한 주사위 랜덤 구현
- 두 명의 사용자 턴 기반 게임 진행
- 뱀 / 사다리 로직을 배열 하나로 단순화하여 구현
- 이동 범위 초과(100번 칸 초과) 예외 처리
- 콘솔 기반 보드 출력 (지그재그 형태)
- `getchar()`를 이용한 턴 대기 처리
